# 🚗 Dashboard de Vendas Geely – 1º Trimestre

Dashboard interativo desenvolvido em HTML/CSS/JavaScript para análise de vendas da concessionária Geely, com dados reais do primeiro trimestre. O painel apresenta métricas-chave, gráficos dinâmicos e filtros interativos para exploração dos dados.

---

## 📊 Visão Geral

Este projeto foi desenvolvido com o objetivo de demonstrar habilidades em **análise de dados**, **visualização interativa** e **desenvolvimento front-end**. O dashboard transforma uma planilha bruta de vendas em insights visuais de fácil interpretação.

---

## 🎯 Indicadores-Chave (KPIs)

O painel exibe quatro métricas principais, atualizadas automaticamente conforme os filtros são aplicados:

| KPI | Descrição |
|-----|-----------|
| **Faturamento Total** | Soma de todas as vendas (em R$) |
| **Unidades Vendidas** | Quantidade total de veículos comercializados |
| **Ticket Médio** | Valor médio por venda |
| **Vendedores Ativos** | Número de vendedores que realizaram ao menos uma venda |

---

## 📈 Gráficos Dinâmicos

Três visualizações gráficas complementam a análise:

### 1. Vendas por Modelo
- **Tipo:** Gráfico de barras
- **Descrição:** Mostra o faturamento gerado por cada modelo de veículo (EX2 PRO, EX2 MAX, EX5 PRO, EX5 MAX)

### 2. Vendas por Loja
- **Tipo:** Gráfico de pizza
- **Descrição:** Distribuição do número de vendas entre as lojas (Gastão Matriz, JD Europa, Campinas)

### 3. Desempenho por Vendedor
- **Tipo:** Gráfico de barras horizontais
- **Descrição:** Ranking dos vendedores pelo faturamento total gerado

---

## 🎛️ Filtros Interativos

O dashboard permite filtrar os dados por três critérios, com atualização em tempo real:

- **Loja** – Seleciona uma das concessionárias
- **Cor do Veículo** – Filtra por cor (Branco, Cinza, Prata, Preto, Verde Pistáchio)
- **Vendedor** – Exibe apenas as vendas de um profissional específico
